Cloudflare introduced temporary accounts for AI agents, allowing ephemeral 60-minute deployments via `wrangler deploy --temporary`. These deployments can be claimed permanently within the window. <summary><span class="hz-item-title">CSSQuake: Classic Quake Recreated Using CSS Rendering</span> <span class="hz-item-score">⭐️ 7.0/10</span></summary>

CSSQuake is a recreation of the original Quake game that renders the game world using CSS, powered by PolyCSS, while still relying on JavaScript for game logic. This project demonstrates the extreme flexibility and creative potential of web technologies, pushing CSS beyond its typical styling role into real-time 3D rendering, which inspires new approaches to web-based game development. The rendering is done entirely with CSS, but JavaScript is required for game logic, input handling, and other runtime tasks. Performance is notably lower than the original software-rendered Quake, even on modern hardware like the Apple M1 Pro.

🔗 [Source](https://cssquake.com/)

hackernews · msalsas · Jun 20, 10:49 · [Discussion](https://news.ycombinator.com/item?id=48608223)

**Background**: The original Quake (1996) used a software renderer with binary space partitioning (BSP) and Gouraud shading to achieve real-time 3D on mid-90s PCs. CSS is a stylesheet language for describing the presentation of web documents, not designed for 3D rendering. CSSQuake repurposes CSS properties to simulate 3D graphics, a technique sometimes called "CSS 3D" or "CSS rendering".

Sean Lynch argues that the primary value of the Model Context Protocol (MCP) is isolating the authentication flow outside the agent's context window, potentially serving as an auth gateway for APIs. This insight reframes MCP's role beyond tool integration to a security boundary mechanism. This perspective highlights a critical security advantage of MCP over traditional skills or CLI approaches, addressing a key challenge in deploying LLM agents: managing authentication without exposing credentials within the agent's context. It could influence how developers design secure agent architectures and API integrations. Lynch suggests that even if MCP only served as an auth gateway for APIs, it would still be a win. The comment references MCP's ability to isolate auth flow outside the agent's context window and potentially out of the harness completely, contrasting with skills/CLI where auth is handled within the agent.

🔗 [Source](https://simonwillison.net/2026/Jun/19/sean-lynch/#atom-everything)

rss · Simon Willison · Jun 19, 22:45

**Background**: The Model Context Protocol (MCP) is an open standard introduced by Anthropic in November 2024 to standardize how AI systems integrate with external tools and data sources. Traditional approaches like skills or CLI require the agent to handle authentication directly, which can expose credentials and increase security risks. MCP provides a standardized interface for tool discovery and invocation, with built-in support for authentication, audit logging, and permission scoping.

<summary><span class="hz-item-title">Datasette Apps: Sandboxed HTML/JS apps inside Datasette</span> <span class="hz-item-score">⭐️ 7.0/10</span></summary>

The datasette-apps plugin allows hosting sandboxed HTML+JavaScript applications inside Datasette that can execute read-only and configured write SQL queries against the underlying data. This plugin transforms Datasette from a data exploration tool into a platform for building custom interactive web applications directly on top of SQLite databases, expanding its use cases for developers and data publishers. Apps run in a sandboxed iframe with `allow-scripts allow-forms` and a CSP header that blocks outbound HTTP requests, preventing data exfiltration. Write queries require pre-configured stored queries.

**Background**: Datasette is an open-source tool for exploring and publishing SQLite databases, with a plugin system that extends its functionality. The sandboxed iframe pattern is a standard web security mechanism that restricts what the embedded content can access, preventing malicious code from stealing data or making network requests.
