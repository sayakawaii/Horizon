Zeroserve 是一款新型零配置 Web 服务器，它利用 eBPF 进行脚本化，作为 nginx 和 Caddy 的替代方案，专注于性能和简洁性。 这种新颖的方法将 eBPF 与 Web 服务器脚本化相结合，有望在不使用传统配置文件的情况下实现高性能、安全且可扩展的请求处理。 Zeroserve 使用 Rust 编写，允许用户用 C 语言编写 eBPF 程序来自定义请求处理。目前它是单线程的，专注于静态文件服务，并计划未来进行改进。

**背景**: eBPF 是一种允许在 Linux 内核中运行沙盒程序的技术，用于网络、可观测性和安全。传统的 Web 服务器如 nginx 使用声明式配置语言，而 Zeroserve 用 eBPF 脚本替代，提供了更大的灵活性。

**背景**: 传统 PC 中 CPU 使用独立 RAM，GPU 使用独立 VRAM，数据需要在两者间复制，可能导致延迟和效率低下。统一内存架构（UMA）允许两个处理器访问同一内存池，简化编程并提升 AI 推理和游戏等任务的性能。英伟达的这一提案标志着其从 GPU 主导地位向消费级 PC CPU 硬件的重大扩展。
