The computer was built by Linn Products, known for the Sondek LP12 turntable, and used capability-based addressing to enforce security at the hardware level. Tagged architectures attach metadata (tags) to each memory word to distinguish data from capabilities, enabling fine-grained protection.

<summary><span class="hz-item-title">Firefox compiled to WebAssembly runs inside a browser</span> <span class="hz-item-score">⭐️ 8.0/10</span></summary>

Puter has compiled Firefox (Gecko engine) to WebAssembly, enabling the full browser to run inside another browser. The demo uses a 233MB gecko.wasm file and proxies all traffic via the Wisp protocol through Puter's servers. This is a significant engineering achievement that demonstrates the feasibility of running a full browser inside a sandboxed WebAssembly environment, with potential applications in browser isolation, edge computing, and secure remote browsing. The use of AI-assisted development (costing ~$25,000 in tokens) also highlights the growing role of LLMs in complex software projects. The project chose Firefox/Gecko for its strong single-process support, which simplifies the WebAssembly compilation. All network traffic is proxied through Puter's server using the Wisp protocol, and the team had to scale servers to handle Hacker News traffic. The demo claims end-to-end encryption, verified by inspecting WebSocket messages.

🔗 [Source](https://simonwillison.net/2026/Jul/16/firefox-in-webassembly/#atom-everything)

rss · Simon Willison · Jul 16, 23:34

**Background**: WebAssembly (WASM) is a low-level binary instruction format that runs in modern web browsers at near-native speed, enabling languages like C++ to be compiled for the web. Firefox's Gecko engine is a complex browser engine that typically runs as a native application; compiling it to WASM requires significant adaptation. The Wisp protocol is a low-overhead protocol for proxying TCP/UDP sockets over a single WebSocket connection, which is necessary because WebAssembly in browsers cannot open raw network sockets.
